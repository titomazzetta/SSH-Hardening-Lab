# Nmap SSH Scan Examples (Lab / Authorized Only)

## Basic banner + version detection
nmap -sV -p 22 <SERVER_IP>

## Default scripts (safe) + version detection
nmap -sC -sV -p 22 <SERVER_IP>

## Enumerate SSH algorithms (useful to validate crypto policy)
nmap --script ssh2-enum-algos -p 22 <SERVER_IP>

## Check supported auth methods (helps confirm key-only vs password vs MFA posture)
nmap --script ssh-auth-methods --script-args="ssh.user=secadmin" -p 22 <SERVER_IP>
